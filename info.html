About TroLab

TroLab is a decentralized experimental lab prototype designed to run experiments, share results, and collaborate with others. It serves as a demo and template for future Tro-based apps.

Design Process

Idea: Create a simple platform to experiment with Tro technology.
Concept: Build a PWA with user-controlled experiments and sharing features.
Build: Develop a lightweight, responsive site with no frameworks.
Deploy: Host on GitHub Pages for free, open-source access.
